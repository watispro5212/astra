TACTICAL SECTORS

Astra v7.0.0 Nova ships 9 fully autonomous intelligence sectors — each engineered to be the best-in-class solution for elite communities.

🛡️

MODERATION SECTOR

A multi-tier enforcement system with automatic case logging, hierarchy protection, and full operative history tracking.

  • Kick, ban, unban, timeout, warn — all with case IDs
  • DM-based warnings with automatic fallback
  • Per-server case lookup and full history view
  • Bulk message purge (1–100, skips 14-day-old messages)
  • Role hierarchy protection prevents abuse
📈

INTELLIGENCE SECTOR

Global XP tracking that rewards authentic engagement. Operatives carry their rank across every Astra-linked community.

  • 15–25 XP per message with 60-second anti-spam cooldown
  • Visual ASCII progress bar toward next level
  • Automatic level-up announcements in channel
  • Global top-10 leaderboard with live username resolution
  • Lookup any operative's rank and XP
💰

FISCAL SECTOR

A complete community economy with multiple income streams, peer-to-peer transfers, and a global wealth leaderboard.

  • Daily allocation: 2,500 credits every 24 hours
  • Work missions: 250–750 cr, 1-hour cooldown
  • High-risk mining: 500–2,500 cr or loss, 30-min cooldown
  • Peer-to-peer credit transfers between operatives
  • Global top-10 wealth leaderboard
🎫

LOGISTICS SECTOR

New in v7

A full private-channel ticket system. Members open tickets, staff are automatically granted access, and the lifecycle is managed entirely in Discord.

  • Private channel per ticket — hidden from everyone else
  • Automatic staff role permission grants via config
  • One active ticket per user prevents spam
  • Add/remove users mid-ticket
  • 5-second grace period on close before deletion
🎉

ENTERTAINMENT SECTOR

New in v7

Host fully automated giveaways with reaction-based entry, configurable winner counts, and a background auto-end service.

  • Timed giveaways (1 minute – 7 days)
  • Up to 10 simultaneous winners per draw
  • Background checker auto-ends expired giveaways
  • Manual end and one-command reroll
  • Bot entries automatically filtered out
👋

WELCOME SECTOR

New in v7

Fully configurable onboarding — custom welcome messages, auto-role assignment, farewell messages, and live preview testing.

  • Custom message with {user} / {server} placeholders
  • Auto-role assignment on member join
  • Separate farewell channel and message
  • Live test command to preview instantly
  • Full config view in one command
🚀

UTILITY SECTOR

Essential productivity tools for members and staff — reminders, community polls, and deep-tissue diagnostic reports.

  • Personal timed reminders delivered in-channel
  • Reaction-based polls with up to 10 options
  • Operative profile with join date and role info
  • Server telemetry — members, channels, boost tier
📊

INFO SECTOR

System-level diagnostics and user intelligence — from full-res avatars to live latency readings and memory metrics.

  • 1024px avatar display for any operative
  • Live stats: uptime, RAM, latency, guild count
  • Detailed user intelligence reports
  • Server snapshot with full configuration data
⚙️

SYSTEM SECTOR

Core operations for administrators — system-wide alerts, command synchronisation, and the Nuclear Protocol server reconstruction.

  • Broadcast alerts to all linked sectors simultaneously
  • Guild / global / clear command sync scopes
  • Apex reconstruction: full channel + role rebuild
  • Vanguard Error Reporter notifies developers in real-time
  • Nova v7.0.0 release summary on demand
