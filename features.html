Features

Everything Astra brings to your server — organized by category.

🛡️ Moderation

Kick & Ban

Remove disruptive members instantly. Bans include a confirmation prompt to prevent accidents, with an option to delete recent message history. All actions are logged as moderation cases.

Warnings

Issue recorded warnings to members. Each warning is stored with a unique case number, reason, and moderator. Users receive a DM notification so they know what they did.

Case History

View a member's full moderation history with the cases command. See every kick, ban, and warning with dates, reasons, and which moderator took action.

Event Logging

Automatic logging for member joins, leaves, bans, message edits, and message deletions. All events post to a configurable log channel with rich embeds.

💬 Community Engagement

Interactive Polls

Create polls with up to 10 options using button-based voting. Results update in real-time with vote counts and percentages. Persistent across bot restarts.

Starboard

The best messages rise to the top. When a message gets enough ⭐ reactions, it's automatically reposted to your starboard channel. Star counts update live as reactions change.

⚙️ Server Management

Role Menus

Let members pick their own roles through interactive button or select menus. Supports unique mode, which removes other roles in the menu when a new one is selected.

Ticket System

A full support ticket system with a launcher panel, private ticket channels, and staff controls. Users click a button to open a ticket, and staff are notified with a control panel to close it.

Server Configuration

Configure logging channels, starboard channels, and star thresholds with simple slash commands. View your current settings at any time with a single command.

🔧 Utility

Reminders

Set personal reminders with flexible time formats like 10m, 1h, or 2d. Astra checks every minute and pings you in the channel when your reminder fires. Manage active reminders anytime.

Bot Info & Latency

Check Astra's response time with the ping command, which shows both gateway and REST latency. The about command gives you key info about the bot's framework and developer.
