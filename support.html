🐛 Reporting a Bug or Anomaly

If you encounter unexpected behavior, the fastest resolution path is the official Discord sector. When reporting, include:

  • › The exact command you ran (e.g. /mod warn)
  • › The error message or unexpected output you received
  • › Your server ID and approximate time of the incident

The Vanguard Error Reporter is active in Nova v7.0.0 — critical failures are automatically DM'd to the development team, so many issues are resolved before users even report them.

🔧 Self-Hosting Astra
Astra is open for self-hosting under the Astra Community License v2.1. Clone the repository, copy .env.example to .env, fill in your DISCORD_TOKEN, DISCORD_CLIENT_ID, and OWNER_ID, then run npm run dev (development) or npm run build && npm start (production). SQLite is used by default — set DATABASE_URL to a PostgreSQL connection string for production deployments.
