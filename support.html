Troubleshooting

Common Issues

Astra isn't responding to commands

  • Ensure Astra has Use Application Commands permission.
  • Verify Astra is online. If not, join the support server for status updates.
  • Try running /ping to check if the bot is responsive.

XP & Leveling aren't working

  • Astra has a 1-minute XP cooldown per user to prevent spamming.
  • Ensure the "Level Notification" setting isn't disabled in your config.
  • Level roles must be below Astra's highest role for her to assign them.

Auto-Moderation is too strict/loose

  • Use /automod_config to recalibrate filters for caps, spam, and links.
  • Members with Manage Messages or Administrator are exempt by default.
  • Check if you have accidentally whitelisted certain roles.

Starboard isn't posting

  • Run /config starboard to set a destination channel.
  • Ensure /config threshold is set to a value greater than 0.
  • Astra excludes self-reactions and bot reactions from the star count.

Tickets aren't creating

  • Run /ticket_setup to configure the category and staff roles.
  • Astra must have Manage Channels in the chosen category.
  • Verify the category isn't at the 50-channel limit.

Still need help?

If none of the above solve your issue, join our support server and describe the problem. Include your server ID and any error messages you see.
