Troubleshooting

Common Issues

Astra isn't responding to commands

  • Make sure Astra is online in your server's member list.
  • Check that Astra has the Use Application Commands permission in the channel you're using.
  • Try running /ping to verify connectivity.
  • If commands don't appear, the bot may need to re-sync. Try kicking and re-inviting Astra.

Astra can't kick/ban members

  • Astra's role must be higher than the target member's highest role in the server role hierarchy.
  • Ensure Astra has the Kick Members or Ban Members permission.
  • Astra cannot moderate the server owner or members with equal/higher roles.

Starboard isn't working

  • Run /config starboard to set the starboard channel.
  • Run /config threshold to set the required number of stars.
  • Stars from bots and the message author are excluded from the count.
  • Make sure Astra can send messages and embed links in the starboard channel.

Tickets aren't creating

  • Run /ticket_setup to configure the ticket category and staff role first.
  • Ensure Astra has Manage Channels permission in the ticket category.
  • Check that the ticket category isn't full (Discord has a 50-channel limit per category).

Still need help?

If none of the above solve your issue, join our support server and describe the problem. Include your server ID and any error messages you see.
