🔧 TROUBLESHOOTING

Commands not appearing?

  • Use /system sync to refresh commands
  • Restart your Discord app completely
  • Check if Astra has permission to use slash commands
  • Wait 1-2 hours for Discord to update globally

Bot not responding?

  • Check bot status above — ensure it's online
  • Verify Astra has "Send Messages" and "Use Slash Commands" permissions
  • Try the command in a different channel
  • Use /system status to check bot health

Economy/money issues?

  • Use /economy balance to check your funds
  • Bank deposits/withdrawals may take a moment to process
  • Report any lost money with transaction details
  • Passive income updates every 6 hours

Permission errors?

  • Staff commands require appropriate server roles
  • Check role hierarchy in server settings
  • Admin commands need Administrator permission
  • Use /info permissions to see your access level
