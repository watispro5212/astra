🚀 Welcome

Welcome to the Astra Help Center. This guide will help you setup your bot and use its AI features.

🛰️ Profile Apps

Astra now supports Profile Installation, allowing you to use the bot's features on your personal Discord account.

  • 🔹 Use Anywhere: Use Astra commands in any server, DM, or Group DM.
  • 🔹 AI in DMs: Message Astra directly to talk to the AI assistant.
  • 🔹 Privacy First: Your interactions are private and locked to your account.

How to install:

  1. Click the "Add to Profile" or "Add to Server" button.
  2. Select "Try it Now" (User Install) on the Discord page.
  3. Authorize the bot to see your profile and use commands.

Quick Start

Adding Astra to your server is easy. Once the bot is there, use these commands to see if it is working.

/system status — Check if the bot is online
/info stats — See server stats

Astra will automatically setup its commands when it joins. If commands don't show up, follow the sync steps below.

AI Assistant

The **AI Assistant** is a smart chat bot that works in your **Direct Messages (DMs)**. It uses a reliable system to make sure it's always ready to help.

Getting an AI Key

To work correctly, Astra needs a Google Gemini API key. If your keys aren't working or you run out of uses, follow these steps:

1. Go to Google AI Studio
2. Create a new API Key (Gemini 1.5 Flash is best)
3. Update the keys in your setup

AI Commands

/ai model — Pick between fast or smart modes
/ai status — Check if the AI is working
/ai reset — Clear the current chat

Users can change how the AI talks (pick models) using the /ai model command. You can pick between Flash (Fast) and Pro (Smart).

Bot Prefix

While Astra mostly uses Slash Commands, you can also use a **Prefix** for quick shortcuts. The default prefix is -.

Default Prefix: -
Usage: -[command] [options]

You can change the server prefix using the /config prefix command (must be an Admin).

Refreshing Commands

Sometimes Discord is slow to show new commands. If a command is missing from your list:

1. Type /system sync
2. Wait for it to finish
3. Restart your Discord app

This refreshes the server's command list and adds any missing features.

Database & Storage

Astra uses cloud storage to keep your data safe. This means your levels and money are always saved and never lost.

• High speed database
• Data is always backed up
• Always saved

Economy & Money

Astra's economy has three main parts: Mining, Shops, and Stocks.

/economy mine — Mine for money (risk included!)
/economy harvest — Collect your daily earnings
/stockmarket market — See live stock prices

Use /economy harvest to get money from your server activity. If you don't see the command, try using /system sync.
