🚀 Quick Start

Adding Astra to your server is easy. Once the bot is in your server, use these commands to check that everything is working.

/system status — Check if the bot is online
/info stats — See server and bot stats

Astra automatically registers its slash commands when it joins. If commands are missing, use /system sync to force a refresh.

⚙️ Server Setup & Configuration

Configure Astra to work perfectly with your server's needs and moderation style.

Essential Permissions

• Send Messages — Required for all bot responses
• Use Slash Commands — Required for command usage
• Embed Links — Required for rich message formatting
• Read Message History — Required for some moderation features
• Manage Messages — Required for purge and moderation
• Manage Roles — Required for role management features

Staff Role Setup

/server_setup staff_role — Set moderator roles
/server_setup admin_role — Set administrator roles
/server_setup log_channel — Set where moderation logs go
/server_setup welcome_channel — Set welcome messages

Use /server_setup commands to configure Astra for your server. These settings are per-server and help customize the bot's behavior.

🛰️ Profile Apps

Astra supports Profile Installation, letting you use the bot's features from your personal Discord account — no server required.

  • 🔹 Use Anywhere: Use Astra commands in any server, DM, or Group DM.
  • 🔹 AI in DMs: Message Astra directly to talk to the built-in AI.
  • 🔹 Privacy First: Your interactions are private and locked to your account.

How to install:

  1. Click the "Add to Profile" button on this website.
  2. Select "Try it Now" (User Install) on the Discord authorization page.
  3. Authorize the bot to access your profile.

🔄 Refreshing Commands

Sometimes Discord is slow to show new slash commands. If a command is missing from your list:

1. Type /system sync
2. Wait a few seconds for it to finish
3. Restart your Discord app if needed

This forces the server's command list to update and adds any missing features.

🤖 AI Assistant

The AI Assistant works in your Direct Messages (DMs). Just send a private message to Astra and the AI will reply instantly. Your conversation stays private.

AI Commands

/ai model — Pick your preferred AI model
/ai status — Check if the AI is working
/ai info — See info about available models

Astra uses free AI models via OpenRouter. If one model is slow or unavailable, the bot automatically switches to a backup.

💰 Economy & Money

Astra's economy has three main parts: earning money, the server shop, and the stock market.

/economy daily — Collect your daily money reward
/economy work — Work for money (1 hour cooldown)
/economy mine — Mine for money (risky!)
/economy harvest — Collect passive income from shop items
/stockmarket market — See live stock prices

The bank keeps your money safe from robbery. Use /economy bank deposit to move money in and /economy bank withdraw to take it out.

🛡️ Moderation Tools

Astra provides comprehensive moderation features to help keep your server safe and organized.

Basic Moderation

/moderation warn — Warn a user with optional reason
/moderation kick — Remove a user from the server
/moderation ban — Ban a user permanently
/moderation unban — Unban a previously banned user
/moderation mute — Temporarily mute a user
/moderation unmute — Remove a user's mute

Advanced Features

/moderation purge — Delete multiple messages at once
/moderation lockdown — Lock a channel temporarily
/moderation slowmode — Set channel slowmode delay
/moderation role — Add/remove roles from users

All moderation actions are logged and can be viewed with /moderation history. The bot requires appropriate permissions to perform these actions.

📈 Leveling & XP System

Astra's leveling system rewards active participation with XP and levels that unlock special perks.

How XP Works

  • 🔹 Earn XP: Send messages, use commands, and participate in voice channels
  • 🔹 Level Up: Reach XP thresholds to advance levels
  • 🔹 Global System: Your level follows you across all servers
  • 🔹 Anti-Abuse: XP rates are balanced to prevent farming

Leveling Commands

/leveling rank — Check your current level and XP
/leveling leaderboard — See top users in the server
/leveling profile — View detailed profile with stats
/leveling rewards — See what you unlock at each level

Higher levels unlock special roles, increased economy rewards, and exclusive commands. Use /leveling config to customize XP rates (staff only).

🛒 Server Shop

The server shop lets you spend money on items that provide passive income and special perks.

Shop Items

• Farms — Generate passive income every 6 hours
• Businesses — Higher income but more expensive
• Luxury Items — Status symbols and bonuses
• Collectibles — Rare items with special effects

Shop Commands

/shop browse — See all available items
/shop buy — Purchase an item
/shop inventory — View your owned items
/shop sell — Sell an item back (at reduced price)
/shop harvest — Collect income from all items

Items provide ongoing passive income. Use /economy harvest to collect earnings. Shop items persist across servers and provide global benefits.

📊 Stock Market

Astra's stock market lets you invest virtual money in stocks that fluctuate based on real market data.

How It Works

  • 🔹 Real Stocks: Based on actual market prices (delayed)
  • 🔹 Virtual Money: Use your economy balance to invest
  • 🔹 Live Updates: Prices update regularly throughout the day
  • 🔹 Portfolio: Track your investments and profits/losses

Trading Commands

/stockmarket market — View current stock prices
/stockmarket buy — Purchase shares of a stock
/stockmarket sell — Sell your shares
/stockmarket portfolio — See your investments
/stockmarket history — View price history for a stock

Stocks are priced in virtual currency matching real market values. This is for entertainment only — not financial advice!

☁️ Database & Storage

Astra uses cloud storage to keep your data safe. Your levels, money, and mod history are always saved — they survive bot restarts and server changes.

• Supports both PostgreSQL (cloud) and SQLite (local)
• Data is never deleted unless you use a reset command
• Economy and leveling data are stored per-user globally

⚡ Bot Prefix

Astra primarily uses Slash Commands, but also supports a message prefix for quick shortcuts. The default prefix is -.

Default Prefix: -
Example: -ping, -stats, -economy

Prefix shortcuts only work in DMs. All full-featured commands use Discord's built-in slash command system (/).
