Quick Start

Deploying Astra to your sector is a streamlined process. Once the bot is present, use the following diagnostic commands to verify system integrity.

/system status
/info stats

The **Titan Engine** will automatically synchronize its tactical assets upon arrival. No manual registration is required.

Interface Protocols

Astra supports dual-mode interaction protocols to ensure maximum compatibility across all terminal types.

🛰️ Slash Commands (Recommended)

Modern, high-fidelity interaction via Discord's native slash command menu. All 12 tactical sectors are accessible here.

🛠️ Prefix Commands (Legacy)

Message-based commands for rapid execution. The default prefix is -, but can be customized per-server.

/prefix new_prefix: !

Admins can use the command above to recalibrate the tactical interface to their preference.

Industrial Finance

Astra's fiscal ecosystem is divided into three primary sectors: Extraction, Industry, and Speculation.

/economy mine
/economy harvest
/stockmarket list

Operatives should use /economy harvest to collect passive income from industrial assets. If commands are missing from your terminal, execute /system sync to force a sector-wide asset refresh.
