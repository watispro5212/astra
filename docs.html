🚀 Quick Start

Adding Astra to your server is easy. Once the bot is in your server, use these commands to check that everything is working.

/system status — Check if the bot is online
/info stats — See server and bot stats

Astra automatically registers its slash commands when it joins. If commands are missing, use /system sync to force a refresh.

🛰️ Profile Apps

Astra supports Profile Installation, letting you use the bot's features from your personal Discord account — no server required.

  • 🔹 Use Anywhere: Use Astra commands in any server, DM, or Group DM.
  • 🔹 AI in DMs: Message Astra directly to talk to the built-in AI.
  • 🔹 Privacy First: Your interactions are private and locked to your account.

How to install:

  1. Click the "Add to Profile" button on this website.
  2. Select "Try it Now" (User Install) on the Discord authorization page.
  3. Authorize the bot to access your profile.

🔄 Refreshing Commands

Sometimes Discord is slow to show new slash commands. If a command is missing from your list:

1. Type /system sync
2. Wait a few seconds for it to finish
3. Restart your Discord app if needed

This forces the server's command list to update and adds any missing features.

🤖 AI Assistant

The AI Assistant works in your Direct Messages (DMs). Just send a private message to Astra and the AI will reply instantly. Your conversation stays private.

AI Commands

/ai model — Pick your preferred AI model
/ai status — Check if the AI is working
/ai info — See info about available models

Astra uses free AI models via OpenRouter. If one model is slow or unavailable, the bot automatically switches to a backup.

💰 Economy & Money

Astra's economy has three main parts: earning money, the server shop, and the stock market.

/economy daily — Collect your daily money reward
/economy work — Work for money (1 hour cooldown)
/economy mine — Mine for money (risky!)
/economy harvest — Collect passive income from shop items
/stockmarket market — See live stock prices

The bank keeps your money safe from robbery. Use /economy bank deposit to move money in and /economy bank withdraw to take it out.

☁️ Database & Storage

Astra uses cloud storage to keep your data safe. Your levels, money, and mod history are always saved — they survive bot restarts and server changes.

• Supports both PostgreSQL (cloud) and SQLite (local)
• Data is never deleted unless you use a reset command
• Economy and leveling data are stored per-user globally

⚡ Bot Prefix

Astra primarily uses Slash Commands, but also supports a message prefix for quick shortcuts. The default prefix is -.

Default Prefix: -
Example: -ping, -stats, -economy

Prefix shortcuts only work in DMs. All full-featured commands use Discord's built-in slash command system (/).
