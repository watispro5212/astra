Changelog

Track every update, feature, and improvement.

v1.0.0 April 18, 2026 Initial Release
  • Moderation: Added /kick, /ban, /warn, and /cases commands with persistent case tracking
  • Polls: Interactive button-based polls with real-time vote tracking and up to 10 options
  • Tickets: Full ticket support system with launcher panels, private channels, and staff controls
  • Starboard: Automatic starboard with configurable threshold and live count updates
  • Reminders: Personal reminders with flexible time parsing (10m, 1h, 2d)
  • Role Menus: Self-assignable role menus with button and select menu styles
  • Event Logging: Automatic logging for joins, leaves, bans, message edits, and deletions
  • Server Configuration: Slash commands for configuring logging channel, starboard, and threshold
  • General: /ping with gateway and REST latency, /about with bot info
  • Architecture: Modular cog-based design with persistent SQLite database, discord.py 2.3+
