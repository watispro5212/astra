Patch v8.1.0

Bug Fix & Stability Update

APRIL 30, 2026

Critical Bug Fixes

  • Startup Crash: Fixed bot failing to load 8 commands due to a missing PROTOCOL constant. Bot would silently skip entire command files.
  • AI System: Fixed all AI database calls using Postgres-style $1/$2 parameters as arrays instead of ? varargs — broke /ai model and usage tracking entirely.
  • Profile Command: Fixed /profile showing 0 money and XP — was querying non-existent columns credits and bank instead of balance and bank_balance.
  • XP Crash: Fixed RangeError: Invalid count value crash in level-up notifications, /info stats, and /system status — progress bar repeat calls could receive negative numbers.
  • PostgreSQL Compat: Replaced all MAX(0, expr) in SQL SET clauses with GREATEST(0, expr) — MAX() is an aggregate function and invalid in this context in PostgreSQL.
  • Giveaway Version: Giveaway footers showed hardcoded v7.0.0 — now uses the live VERSION constant.

Leveling Rework

  • New XP Formula: Replaced linear (level+1)×500 with quadratic 25n²+250n+500. Leveling up 10× too fast is now fixed — level 10 now requires real sustained activity.
  • Slower XP Rate: Reduced XP per message from 15–25 to 10–20 (avg 15). Cooldown raised from 60s to 90s.
  • Blacklist Enforcement: Blacklisted users no longer earn XP.
  • Formula Sync: /leveling rank progress bar now uses the same formula as the XP handler.

DM Commands — New

  • Slash Commands in DMs: Fixed — commands now always sync globally so they appear in DMs. Previously dev-mode guild sync wiped global commands, making DMs slash-command-free.
  • DM Prefix Commands: Added full -prefix command system in DMs: -balance, -daily, -work, -mine, -gamble, -coinflip, -slots, -rank, -ping, -help.
  • /info help: New subcommand — bot activity says /info help, now it actually works and shows the full command list.

AI Improvements

  • Model Rotation on Rate Limit: When a model hits a 429 upstream rate limit, the bot now tries every fallback model in order before giving up — instead of just rotating API keys.
  • Clean Error Messages: Raw OpenRouter JSON error blobs no longer leak to users. All errors show friendly human-readable messages.
  • Startup DM: Bot now DMs the owner directly when it comes online (in addition to the status webhook).

Website Overhaul

  • New Homepage: Redesigned hero, stats band, 9-cell feature grid, and live terminal demo showing DM commands.
  • Full Commands Page: Every command now listed with accurate descriptions, permission badges, and DM support indicators.
  • HTML Bug Fixes: Removed all duplicate class= attributes across all pages, fixed broken docs sidebar, moved orphaned FAQ items into proper containers, fixed unrendered **markdown** bold text.
  • CSS Fixes: Added missing --color-primary-light and --grad-quantum variables. Removed duplicate utility rules.
