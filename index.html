Features

Everything your server needs

From moderation to community engagement — Astra handles it all with a clean, intuitive experience.

🛡️

Moderation

Kick, ban, and warn with full case tracking. Every action is logged with a case number for accountability.

📊

Interactive Polls

Create button-based polls with real-time vote tracking. Up to 10 options per poll with persistent results.

🎫

Ticket System

Private support channels with one-click setup. Staff get notified instantly, and tickets stay organized.

⭐

Starboard

Highlight the best messages in your community automatically. Configurable star threshold and live count updates.

⏰

Reminders

Set personal reminders with flexible time formats. Astra checks every minute and pings you when it's time.

🏷️

Role Menus

Self-assignable roles with button or dropdown menus. Supports unique mode for mutually exclusive roles.

Getting Started

Up and running in minutes

No complex setup required. Astra works out of the box with simple slash commands.

1

Invite Astra

Add Astra to your server with one click. Grant the permissions it needs to do its job.

2

Configure

Use /config to set your logging channel, starboard, and ticket system. Takes seconds.

3

Enjoy

Start using slash commands. Astra handles the rest — moderation, polls, reminders, and more.
