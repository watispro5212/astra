Privacy Policy

Last updated: April 18, 2026 — Effective for Astra v4.0.0 and all subsequent versions.

1. Overview

Astra ("the Bot") is committed to protecting your privacy. This policy explains what data the Bot collects, why it's collected, how it's stored, and your rights regarding that data. By using the Bot, you consent to the practices described here.

2. Data We Collect

Astra collects only what is strictly necessary for its features to function. This includes:

Server Data

  • Guild (Server) IDs — Used to store per-server configuration (logging channels, automod settings, ticket setup, etc.)
  • Channel IDs & Role IDs — Used for feature configuration (e.g., log channels, welcome channels, ticket categories)

User Data

  • User IDs — Required to track XP, economy balances, reminders, warnings, birthdays, AFK status, and invite counts
  • Usernames & Avatars — Fetched from Discord's API in real-time for display purposes; not permanently stored
  • Join dates — Used for server statistics and invite tracking; sourced from Discord's API

Activity Data

  • Message metadata — Message IDs and timestamps are stored for starboard and poll tracking; message content is not permanently stored
  • Automod — Messages are checked in real-time against configured rules. Content that triggers a rule is logged briefly for moderator review but is not stored long-term
  • Moderation cases — Action type, reason, moderator ID, target ID, and timestamp for audit trails
  • Economy transactions — Coin amounts and action types (no personal financial data)
  • Invite data — Invite codes and join/leave events to support invite tracking leaderboards

3. How We Use Your Data

Data collected by Astra is used exclusively to provide and improve the Bot's features:

  • Persisting server configuration across restarts
  • Tracking XP, levels, economy balances, and leaderboards
  • Delivering reminders, birthday announcements, and giveaway results
  • Generating moderation case logs and audit trails
  • Enabling invite tracking and anti-raid protection

We do not sell, share, rent, or otherwise transfer your data to third parties for marketing or commercial purposes.

4. Data Storage & Security

All data is stored in a SQLite database on the server where Astra is hosted. If you are using the official hosted instance, data is stored on the Developer's private server. Access to the database is restricted to the Bot process and the Developer.

We use reasonable technical measures to protect stored data, but no system is 100% secure. Use of the Bot is at your own risk.

5. Data Retention

Data is retained for as long as it is needed for the feature it supports:

  • Server config — Retained until the Bot is removed from the server
  • User data (XP, economy, birthdays) — Retained until a deletion request is submitted or the Bot is removed
  • Moderation cases — Retained indefinitely for audit purposes
  • Reminders — Deleted automatically once delivered
  • AFK status — Deleted automatically when the user sends a message

6. Third-Party Services

Astra interacts with Discord's API to function. Discord's own Privacy Policy governs how Discord handles your data. Patreon supporters' data is governed by Patreon's Privacy Policy. Astra does not use any other third-party data processors.

7. Your Rights

You have the right to:

  • Request data deletion — Contact us to have your personal data removed from the database
  • Request a data export — Request a copy of data stored about you
  • Opt out — Stop using the Bot at any time; your data will remain until you request deletion

To exercise these rights, contact the Developer via the Support Server or GitHub.

8. Children's Privacy

Astra is not directed at individuals under the age of 13 (or the minimum age required by your jurisdiction). If you believe a minor has provided personal data through the Bot, please contact us immediately.

9. Changes to This Policy

This policy may be updated from time to time. The "Last updated" date at the top reflects the most recent revision. Continued use of the Bot after changes constitutes acceptance of the updated policy.

10. Contact

For privacy-related inquiries, contact the Developer via the Astra Support Server or GitHub.
