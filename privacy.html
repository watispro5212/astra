Privacy Policy

Last updated: April 18, 2026

Introduction

This privacy policy explains what data Astra ("the bot") collects, why it collects it, and how it is stored and used. By adding Astra to your Discord server, you agree to the practices described in this policy.

Data We Collect

Astra collects and stores the minimum data necessary to provide its features. This includes:

Server Configuration

  • Guild (server) IDs
  • Channel IDs for logging and starboard
  • Role IDs for ticket staff and role menus
  • Category IDs for ticket channels
  • Starboard threshold settings

Moderation Data

  • User IDs of moderated members and moderators
  • Action type (kick, ban, warning)
  • Reason text provided by moderators
  • Timestamps of moderation actions
  • Case numbers

Feature Data

  • Polls: Poll questions, options, voter user IDs, and vote counts
  • Reminders: User IDs, channel IDs, reminder messages, and trigger times
  • Starboard: Message IDs, starboard post IDs, star counts
  • Tickets: Ticket configuration (category and staff role IDs)
  • Role Menus: Message IDs, role IDs, menu options, and label text

What We Do Not Collect

  • Message content (except user-provided fields like moderation reasons, poll questions, and reminder messages)
  • Direct message content
  • User passwords or authentication tokens
  • Personal information beyond Discord user IDs
  • Voice channel data

Why We Collect Data

All data is collected solely to provide Astra's features. For example:

  • Moderation cases are stored so staff can review a member's history
  • Reminders require storing your message and trigger time
  • Polls require tracking votes to display accurate results
  • Server configs are stored so your settings persist between restarts

Data Storage

Data is stored in a SQLite database on the server hosting Astra. The database is not shared with any third parties. Access to the database is restricted to the bot's operators.

Data Sharing

We do not sell, share, or distribute any collected data to third parties. Data is only used internally to provide Astra's functionality.

Data Retention

Data is retained as long as it is needed for its intended purpose:

  • Moderation cases are stored indefinitely for accountability
  • Reminders are deleted after they fire
  • Poll data persists for the lifetime of the poll
  • Server configs persist until manually changed or the bot is removed

Data Deletion

You can request deletion of your data by contacting us in our support server. We will remove all data associated with your server ID or user ID upon verified request. Removing Astra from your server will stop future data collection, but existing records remain until a deletion request is processed.

Children's Privacy

Astra does not knowingly collect data from children under 13. Discord's Terms of Service already require users to be at least 13 years old.

Changes to This Policy

We may update this privacy policy from time to time. Significant changes will be announced in the support server. Continued use of Astra after changes constitutes acceptance of the updated policy.

Contact

For questions about this privacy policy or data requests, join our support server and reach out to a team member.
