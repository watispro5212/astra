v4.0.0 Command Reference

All 50+ commands

Every slash command, described and categorized. Use the search to find what you need.

🛡️ Moderation

/mod kick
Kick a member from the server with an optional reason.
Mod
/mod ban
Ban a member with confirmation dialog. Optionally delete messages.
Mod
/mod mute
Timeout a member for a duration (e.g. 10m, 1h, 2d).
Mod
/mod unmute
Remove a member's timeout early.
Mod
/mod unban
Unban a user by their ID.
Mod
/mod history
View moderation history for a user or a specific case.
Mod
/mod appeal
Mark a case as appealed with a reason.
Admin
/mod export
Export all moderation cases as JSON or CSV.
Admin
/warn
Issue a formal warning to a member. Sends DM to the warned user.
Modv3
/warnings
View all active warnings for a member.
Modv3
/delwarn
Remove a specific warning by its ID.
Modv3
/clearwarns
Clear all warnings for a member.
Adminv3
/purge
Delete 1–100 messages from the current channel.
Mod
/slowmode
Set a slowmode delay in seconds. 0 to disable.
Mod
/lock
Lock a channel so members cannot send messages.
Modv3
/unlock
Unlock a previously locked channel.
Modv3
/nick
Change or reset a member's nickname.
Mod

🚨 Anti-Raid

/antiraid config
Configure detection threshold, time window, action (kick/ban), and alert channel.
Adminv3
/antiraid unlock
Lift an active lockdown and resume normal server operation.
Adminv3
/antiraid status
View current anti-raid protection status and settings.
Modv3
