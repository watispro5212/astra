Commands

Complete reference for all Astra slash commands. Professional tools at your fingertips.

🔧 General

/ping
Check the bot's response time and API latency.
Everyone
/about
Information about Astra's development, version, and owner.
Everyone
/stats user
View detailed Discord statistics for yourself or another member.
Everyone

🛡️ Advanced Moderation

/mute
Timeout a member for a specified duration (e.g., 10m, 1h).
Moderate Members
/unmute
Remove a timeout from a member early.
Moderate Members
/purge
Mass delete up to 100 recent messages in a channel.
Manage Messages
/slowmode
Set the slowmode delay for a channel.
Manage Channels
/kick
Remove a member from the server and log the case.
Kick Members
/ban
Ban a member with optional message history deletion.
Ban Members
/unban
Unban a user by their Discord ID.
Ban Members
/warn
Issue a warning. The user is DM'd and a case is created.
Manage Messages
/cases
View a user's full moderation and warning history.
Manage Messages

🏆 Engagement

/rank
View your current XP, level, and progress to the next rank.
Everyone
/leaderboard
View the top 10 most active members in the server.
Everyone
/giveaway
Start a persistent, button-based giveaway.
Manage Server
/reroll
Pick new winners for a finished giveaway.
Manage Server

🛡️ Auto-Moderation

/automod_config
Configure automated filters for spam, links, invites, and caps.
Administrator

📊 Utility

/poll
Create a button-based poll. Supports timed and anonymous voting.
Everyone
/remind
Set a personal reminder (e.g., 10m, 1h).
Everyone
/reminders
Manage and list your active reminders.
Everyone

🎫 Support & Roles

/ticket_setup
Configure ticket categories and log channels for the support system.
Administrator
/ticket_panel
Deploy a persistent "Open Ticket" panel.
Administrator
/role-menu create
Create self-assignable role menus with buttons/dropdowns.
Manage Roles

⚙️ Server Tools

/voice hub
Manage "Join-to-Create" temporary voice channel hubs.
Administrator
/welcome_setup
Configure join/leave messages and automated roles.
Administrator
/config view
View current server settings for logging, starboard, and more.
Manage Server
