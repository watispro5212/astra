ARCHIVAL FAQ

Instant intelligence for common questions about the Astra Titan v7.5.0 engine.

General Deployment

What is Astra Titan v7.5.0?

↓
Astra is an enterprise-grade Discord bot with 12 tactical sectors covering moderation, leveling, economy, support tickets, giveaways, welcome systems, and industrial commerce. v7.5.0 'Titan' is the latest high-fidelity build, optimized for performance and security.

Is Astra free to use?

↓
Yes. All features in Titan v7.5.0 are available to every server at no cost. Note: Astra is proprietary software licensed under the APL v2.5. Self-hosting is not supported to ensure centralized security and reliability.
Moderation Protocols

How do I issue a tactical warning?

↓
Execute /mod warn @user [reason]. The warning is logged in the neural database and the target operative is notified via private transmission. You can review an operative's full history with /mod history @user.

What is the Apex Nuclear Protocol?

↓
The /setup_server nuclear mode completely purges all channels and roles, rebuilding a professional server hierarchy from scratch. This is an irreversible command intended only for initial server deployment.
Cloud Infrastructure

Why was my data resetting before?

↓
Legacy versions of Astra utilized local SQLite storage, which was purged during deployment cycles. **Titan v7.5.0** has migrated to **Supabase Cloud (PostgreSQL)**, ensuring your intelligence and financial assets are stored permanently in a redundant, high-performance database.
Neural Intelligence

How do I access the AI Sentinel?

↓
The Sentinel operates exclusively in **Secure Direct Messages (DMs)**. Simply send a private transmission to Astra and the neural matrix will engage automatically. Note: This feature is isolated from public server channels for maximum operative privacy.

Can I choose different AI models?

↓
Yes. Use the /ai model command to recalibrate the Sentinel. You can choose between **Titan Flash** (optimized for speed/efficiency) or **Titan Pro** (optimized for complex reasoning and deep analysis).

Why are some protocols (like /economy harvest) missing?

↓
Due to Discord's global command caching, some protocols may experience indexing delays. If a command does not appear in your terminal, an administrator must execute the /system sync protocol to force a sector-wide re-indexing of tactical assets.

How do I recalibrate the command prefix?

↓
Astra supports dual-mode interaction. The default prefix for message-based commands is -. Administrators can change this at any time by executing the /prefix set command.
