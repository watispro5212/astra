Click any "Invite Astra" button on this site, or use the direct invite link. You'll need the Manage Server permission on the target server. Astra requests Administrator by default for full functionality — you can downscope this but some features may not work.
Astra requests Administrator for the simplest setup. If you prefer minimal permissions, the core requirements are: Manage Messages, Manage Channels, Manage Roles, Kick/Ban Members, Moderate Members, Send Messages, Embed Links, Read Message History, View Channels, and Add Reactions. Some modules (like anti-raid lockdown) require Administrator to modify channel permissions at scale.
Slash commands can take up to 1 hour to propagate globally after a new deployment. If you're self-hosting, make sure to run /dev sync (owner-only) or set a GUILD_ID in your .env to sync instantly to a specific server. Make sure Astra has the applications.commands scope in its invite link.
Yes — all core features are completely free. Patreon supporters unlock cosmetic perks like custom embed colors, higher XP multipliers, and reduced cooldowns, but no essential functionality is paywalled.
The economy is enabled by default. Members can immediately use /economy daily and /economy work to earn coins. To create shop items, use /economy additem (requires Manage Server). To view the shop, use /economy shop.
Run /antiraid config enabled:True to activate it. You can set the join threshold (default 10 joins), time window (default 10 seconds), alert channel, and the action taken on raiders (kick or ban). Use /antiraid unlock to lift a lockdown after a raid is resolved.
First, run /suggestion config channel:#your-channel to set up the suggestion channel. Members then use /suggest to submit ideas — Astra posts an embed with 👍/👎 reactions. Staff can use /suggestion approve, /suggestion deny, or /suggestion implement to update the status. The suggestion author gets a DM whenever their suggestion is reviewed.
Use /birthday config channel:#announcements to set where birthdays are announced. Optionally add a role: that gets assigned on someone's birthday. Members set their own birthday with /birthday set. Astra checks at 9 AM UTC daily and announces any birthdays for that day.
Yes! Use /backup export to download a JSON file containing all your server's Astra configuration (channels, roles, automod settings, ticket config, economy settings, etc.). Use /backup import to restore from that file on the same server or a different one.
Astra stores only what's necessary to function: server configuration, moderation cases, XP data, economy balances, reminders, and similar feature data. No message content is permanently stored — automod checks messages in real-time and discards them. See the Privacy Policy for full details.
Clone the repository, install dependencies with pip install -r requirements.txt, create a .env file with your DISCORD_TOKEN, and run python main.py. Python 3.11+ required. Enable all three Privileged Gateway Intents (Members, Message Content, Presence) in the Discord Developer Portal. See the README for full setup instructions.
