General Deployment

What is Astra Titan v7.5.0?

↓
Astra is an enterprise-grade Discord bot with 12 tactical sectors covering moderation, leveling, economy, support tickets, giveaways, welcome systems, and industrial commerce. v7.5.0 'Titan' is the latest high-fidelity build, optimized for performance and security.

Is Astra free to use?

↓
Yes. All features in Titan v7.5.0 are available to every server at no cost. Note: Astra is proprietary software licensed under the APL v2.5. Self-hosting is not supported to ensure centralized security and reliability.
Moderation Protocols

How do I issue a tactical warning?

↓
Execute /mod warn @user [reason]. The warning is logged in the neural database and the target operative is notified via private transmission. You can review an operative's full history with /mod history @user.

What is the Apex Nuclear Protocol?

↓
The /setup_server nuclear mode completely purges all channels and roles, rebuilding a professional server hierarchy from scratch. This is an irreversible command intended only for initial server deployment.
Economy & Commerce

How does the Industrial Hub work?

The Industrial Hub (Shop) allows administrators to create a custom marketplace. Members can earn credits via /economy daily and /economy work, then trade them for roles, items, or consumables.

How does the Industrial Hub work?

↓
The Industrial Hub allows administrators to deploy a custom marketplace. Members earn credits via work missions and daily allocations, which can be traded for roles, inventory items, or tactical consumables.
