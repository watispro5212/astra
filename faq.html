FREQUENTLY ASKED QUESTIONS

Quick answers to common questions about the Astra bot.

General Info

What is Astra?

↓
Astra is a powerful Discord bot with many features like moderation, leveling, economy, support tickets, giveaways, and welcome systems. It's built to be simple to use while keeping your server safe and fun.

Is Astra free to use?

↓
Yes. All features of Astra are available to every server at no cost. We don't support self-hosting to make sure every server has the most secure and reliable version of the bot.
Moderation

How do I warn a user?

↓
Use the command /mod warn @user [reason]. The warning is saved in our database and the user is sent a DM to let them know. You can see a user's warnings with /mod history @user.

What does the server setup command do?

↓
The /setup_server command deletes all current channels and roles and rebuilds a professional server setup from scratch. Warning: This cannot be undone, so only use it if you want to start fresh!
Database & Saving

Why was my data resetting before?

↓
Older versions of Astra used a different way to save data that sometimes got lost. Astra now uses a cloud database to make sure your levels and money are saved forever and are always accessible.
AI Assistant

How do I talk to the AI?

↓
The AI works in Direct Messages (DMs). Just send a private message to Astra and the AI will reply to you. This is kept private so no one else can see your conversation.

Can I choose different AI models?

↓
Yes. Use the /ai model command to change which AI you use. You can pick between different models depending on whether you want faster or smarter responses.
Commands & Setup

Why are some commands missing?

↓
Sometimes Discord takes a little while to show new commands. If you don't see a command, an admin can use /system sync to refresh them for the whole server.

How do I change the bot prefix?

↓
Astra uses slash commands by default, but also supports the - prefix for DM shortcuts. All server commands use Discord's slash command system.
