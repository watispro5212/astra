Commands

All of Astra's slash commands at a glance. Every command uses Discord's native slash command system.

🔧 General

/ping
Check the bot's gateway and REST latency.
Everyone
/about
View information about Astra — framework, developer, and version details.
Everyone

🛡️ Moderation

/kick
Kick a member from the server. Logs the action as a moderation case.
Kick Members
/ban
Ban a member with a confirmation prompt. Optionally delete recent message history (0–7 days).
Ban Members
/warn
Issue a recorded warning to a member. The user is notified via DM.
Manage Messages
/cases
View moderation history for a specific user — all kicks, bans, and warnings.
Manage Messages

📊 Polls

/poll
Create an interactive poll with up to 10 comma-separated options. Results are tracked with button-based voting.
Everyone

⏰ Reminders

/remind
Set a personal reminder. Supports flexible time formats like 10m, 1h, 2d.
Everyone
/reminders
List your active reminders with trigger times and messages.
Everyone

🏷️ Role Menus

/role-menu create
Create a self-assignable role menu with buttons or a select dropdown. Format: Label:@Role, Label2:@Role2.
Manage Roles

🎫 Tickets

/ticket_setup
Configure the ticket category and staff role for your server's support system.
Administrator
/ticket_panel
Send the "Open Ticket" button panel to a specified channel.
Administrator

⚙️ Server Configuration

/config logging
Set the channel where Astra logs server events (joins, leaves, bans, edits, deletes).
Administrator
/config starboard
Set the starboard channel where starred messages are reposted.
Administrator
/config threshold
Set the number of ⭐ reactions required for a message to appear on the starboard.
Administrator
/config view
View the current server configuration — logging channel, starboard channel, and threshold.
Manage Server
